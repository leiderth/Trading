## 2️⃣ BINANCE (CRYPTO REAL)

### **Descripción:**
- Exchange de criptomonedas líder mundial
- API oficial de Binance
- Trading real con dinero real

### **Mercados:**
- BTC/USDT, ETH/USDT, BNB/USDT
- SOL/USDT, ADA/USDT, XRP/USDT
- 500+ pares de trading

### **Modos:**

#### **Testnet (Recomendado para empezar):**
- Dinero virtual
- Funciona igual que Binance real
- Gratis e ilimitado
- Balance inicial: 10 BTC + 100,000 USDT

#### **Real:**
- Dinero real
- Requiere depósito
- Comisiones: 0.1%

### **Cómo Conectar:**

#### **TESTNET (Pruebas):**

1. **Crear cuenta Testnet:**
   - Ir a: https://testnet.binance.vision/
   - Login con GitHub
   - Copiar API Key y Secret

2. **En la app:**
   - Click "🔌 Conectar Broker"
   - Seleccionar "Binance (Crypto)"
   - Pegar API Key y Secret
   - ✅ Marcar "Usar Testnet"
   - Click "Conectar"

#### **REAL (Producción):**

1. **Crear cuenta Binance:**
   - Ir a: https://www.binance.com/
   - Registrarse
   - Verificar identidad (KYC)

2. **Generar API Key:**
   - API Management → Create API
   - Habilitar: Spot Trading
   - Deshabilitar: Withdrawals
   - Copiar API Key y Secret

3. **En la app:**
   - Click "🔌 Conectar Broker"
   - Seleccionar "Binance (Crypto)"
   - Pegar API Key y Secret
   - ❌ Desmarcar "Usar Testnet"
   - Click "Conectar"

### **Ventajas:**
- ✅ API oficial
- ✅ Alta liquidez
- ✅ Comisiones bajas (0.1%)
- ✅ 500+ pares
- ✅ Testnet gratuito
- ✅ Trading 24/7

### **Desventajas:**
- ❌ Solo criptomonedas
- ❌ Requiere verificación KYC
- ❌ Volatilidad alta

### **Comisiones:**
- **Maker:** 0.1%
- **Taker:** 0.1%
- **Con BNB:** 0.075%

### **Mínimos:**
- BTC/USDT: 0.00001 BTC (~$0.50)
- ETH/USDT: 0.0001 ETH (~$0.30)
